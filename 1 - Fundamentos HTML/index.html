Atributos

Atributos são como entrada de parâmetros para uma determinada tag, por exemplo:

<a href=""></a>
<img src="" alt="">

No href da tag a colocamos o link para direcionar para uma determinada página, enquanto o src da img se torna o local de onde se encontra a fonte da imagem

Atributos Booleanos

Os atributos booleanos são atributos que recebem verdadeiros ou falsos, por exemplo

<h2 hidden> Title </h2>

Esse hidden fará com que o h2 fique invisivel

Atributo ID

Cada tag pode ser aplicada um ID, uma espécie de RG da tag, deve único e não pode começar com números

<div id="container"> </div>
<h2 id="titulo"> </h2>

Atributo Class

O atributo class é uma classificação de tags, onde consigo compartilhar a mesma classe para diversas tags

<div class="produtos" > </div>
<div class="produtos" > </div>

Atributo data

Esse atributo serve para colocarmos um nome em um novo atributo para podermos resgatarmos posteriormente no CSS ou no Javascript, aplicamos o "data-" a partir do "-"" podemos colocar um nome qualquer, evite colocar números ou carácteres especiais

<div data-qualquer-coisa="1234" > </div>

Atributo style

Essa tag serve para estilizar o conteudo da tag, evite usar essa tag por ser específica e muito ruim para realizar manutenção em grande escala

<div style="color:blue;" > </div>

Elementos de Conteúdo

Semântica

A semântica tem um significado para cada tag, seria possível sim utilizar uma única tag para realizar tudo, porém ao usar tags que façam sentido fica mais fácil de fazer sua leitura para manutenções e até mesmo para os motores de busca. Basicamente temos tags para cada aplicação, como o título temo o h1, o parágrafo temos o p, para imagem temos img e assim vai

Títulos e Parágrafos

São usados para separar quando vamos mencionar o título e seu parágrafo

<h1 class="title" > Sobre mim </h1>
<p class="descricao" > Lorem ipsum, dolor sit amet consectetur adipisicing elit. Quas fugiat totam perspiciatis a culpa impedit nulla molestiae iusto, voluptates similique odit, commodi pariatur vero voluptatum dolorum, soluta id tempora quisquam. </p>

Formatação Básica de Textos

Podemos dar relevancias com os textos usando negrito, italico, marcado ou até riscadp

Utilize as tags como:

  • strong - Para deixar em negrito
  • em - Para deixar em italico
  • mark - Para deixar em marcado como marca texto
  • s - Para deixar o texto riscado

Listas Ordenadas e Não Ordenadas

As listas ordenadas são listas que possuem ordem de sequência como 1,2 e 3. Enquanto as não ordenadas, não tem importancia em que posição estão sendo colocadas

  • Item 1
  • Item 2
  • Item 3
  1. Item 1
  2. Item 2
  3. Item 3

Representação do código dentro do HTML

Podemos utilizar as tags code e a tag pre

<code> serve colocar o código em si, podemos verificar que muda até sua fonte 

            <pre> serve para utilizar identação levando em conta os espaços que temos dentro dessa tag

Hiperlink

Os hiperlinks são links externos que podemos acessá-los bastando colocar em seu atributo href o local de uma pagina específica. Para abrirmos o link em outra aba podemos incluir o target = "_blank"

        <a href="www.google.com" target="_blank"> Google </a>

Imagem

Para adicionar imagem coloque a tag <img> nos parâmetros vamos ter o alt que é uma descrição da imagem em caso de não carregar, o src que é onde a imagem está, se está no computador ou em um link externo e por fim o widht e height que definema largura e a altura da imagem 

        
            <img src="" alt="" widht=200px height=200px />
