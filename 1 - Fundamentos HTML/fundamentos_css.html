Margin

A margin serve para realizarmos distanciamento de outros containers, podemos configurar suas laterais de diversas maneiras, podendo ser:

  • Todas as laterais com único valor
  • Valores distintos para cada lado
  • Utilizando apenas dois valores para vertical e horizontal
Além disso, devemos tomar cuidado com o margin collapsing que basicamente se tivermos dois containers um abaixo do outro e aplicarmos um margin-top ele não somará a margin entre os containers, manterá o mesmo valor

Block
Inline

      border: 1px solid red;
      margin: 20px;
    }

Padding

Não muito diferente do margin o padding faz o distanciamento dentro do prórprio container enquanto o margin faz fora.

Seus valores funcionam iguais do margin onde podemos:

  • Todas as laterais com único valor
  • Valores distintos para cada lado
  • Utilizando apenas dois valores para vertical e horizontal

Block
Inline 
  
.padding{

  width: 100px;
  border: 1px solid green;
  padding: 5px 10px
}

Box-Sizing

O box-sizing serve para dizer se a largura ou altura do container deve levar em consideração o padding adicionado. O que acontece é que quando adicionamos padding o container tende a aumentar seu tamanho para deixar de acordo com o padding adicionado. Se optarmos a utilizado do border-box: box-sizing o container se manterá do mesmo tamanho

Bloco

.block_sem_box_sizing{
  border: 1px solid red;
  width: 100px;
  height: 100px;
  padding: 50px;
  
}
Bloco
  
.block_box_sizing{
  box-sizing: border-box;
  border: 1px solid red;
  width: 100px;
  height: 100px;
  padding: 50px;
}

Font-size

Font-size define basicamente o tamanho das letras, seus valores podem ser valores absolutos quanto relativos. Podem ser executadas por px, em, rem, % e dentre outros. Os tamanhos relativos como em, vão pegar valores do container pai, já o rem vai pegar o tamanho do root, que é 16px
