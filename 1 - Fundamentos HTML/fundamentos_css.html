Fundamentos do CSS

Box model

O box model é uma caixa nele se tem propriedades, como o conteudo principal, largura e algura, preenchimento interno, bordas, margem

Display

Display significa apresentação, existem tag tipo block são tags que preenchem todo o espaço enquanto o tipo inline são blocos que se comportam como caixas uma ao lado da outra
