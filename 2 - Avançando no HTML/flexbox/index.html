CSS Flexbox

Conceitos

No flexbox quando usamos display: flebox temos uma ideia que o container e os items vão trabalhar juntos, onde dentro o container terá eixos main e cross, sendo main o eixo X e o cross eixo Y

Sabendo disso, temos em cada eixo o start e o end, ou seja, onde cada eixo começa ou termina

Além disso, podemos também determinar tamanho dos items e containers

Alignment & Spaces

justify-content

O justify-content é um atributo que faz alinhamentos dentro do container, manipulando a posição dos itens

flex-start

Por padrão o justify-content aplicado é o flex-start onde temos nossos elementos começando no start do eixo main. Se invertermos o eixo com flex-direction: row-reverse o start passa a ser o end, o end passa a ser o start
