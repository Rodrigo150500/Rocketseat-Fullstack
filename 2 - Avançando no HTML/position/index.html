Introdução

No CSS existe uma propriedade position que serve para posicionar elementos dentro do arquivo html. Existem diversos tipos de position, sendo elas:

  • static
  • relative
  • absolute
  • fixed
  • sticky

Ao usar algun desses positions liberamos o acesso a algumas props, sendo:
  • top, bottom
  • left, right
  • inset
  • z-index

Static

A propriedade static é o padrão quando não aplicamos o position explicitamente, então por padrão o static é aplicado.

Relative

O quando usamos o position: relative podemos reposicionar o elemento para outros pontos. Ela possui alguns pontos como:

  • offset relativo a si mesmo - Reposicionamos a partir de onde o nosso elemento se encontra no próprio corpo. Ou seja, o ponto de origem fica onde o elemento está localizado no html
  • normal flow - O normal flow é o fluxo de leitura do html. Então a formação dos elementos é cima para baixo, ao usarmos essa propriedade ela respeita o normal flow sem quebrar o fluxo.
  • stacking context - O stacking context é como um empilhamento, basicamente é um eixo Z para mostrar quem está na frente de qual elemento, são como se fosse camadas que podem estar na mesma altura ou acima/abaixo.

Exemplo

#relative{ position: relative; top: 20px; left: 20px; }

Absolute

O position:absolute usamos para posicionar ao longo da nossa página, podendo posicionar em qualquer ponto da tela pois não respeita o normal flow ou trabalhar apenas dentro de um conteinar

  • Fica em um determinado local na tela - Porém relativo ao containing block próximo ou initial. O relativo ao initial significa que o elemento fica a disposição de toda parte vísivel, enquanto próximo ao containing ele fica apenas em uma área para ser posicionado
  • !normal flow - não respeita o normal flow saindo da sua posição de acordo com o HTML
  • stacking context - Determina a altura no eixo Z
#container-absolute{ position: relative; border: 1px solid #000 } #absolute{ position: absolute; top: 20px; right: 20px; }

Fixed

O position: fixed tem como principal função de fixar na tela mesmo scrollando a tela para cima ou para baixo. Fica tipo o whatsapp nos cantos dos sites

  • fixo na tela relativo ao containing block
  • !normal flow - não participa do normal flow, ou seja, não fica no lugar posicionado como normalmente deixamos no corpo do HTML
  • stacking context - libera o acesso para configurarmos sua altura em relação ao eixo Z
#fixed{ position: fixed; bottom: 20px; right: 20px; } right: 20px;

sticky

O position: sticky ele vai ter um efeito de grudar na tela de acordo com o top. Basicamente nesse posição é necessário ter uma barra de rolagem do próprio body ou de um container pai

  • relativo e fixo - relativo ao elemento pai que tenha mecanismo scroll - overflow
  • normal flow - respeita seu posicionamento no corpo HTML
  • stacking context - libera a configuração do eixo Z
#container-sticky{ border: 1px solid #000; overflow: scroll; height: 150px; } #sticky{ position: sticky; top: 30px; z-index: -1; }

Prop inset

Essa propriedade basicamente serve para aplicar todos os posicionamento de top, botom, left e right ao mesmo tempo. Nesse exemplo aplica apenas no top e no left, pois não tem como aplicar em todos ao mesmo tempo

#inset{ position: relative; inset: 20px }
