Function

Podemos aplicar os estilos de utilizando filtros nos elementos

  • not()
  • has()

not()

Aplicamos quando não o elemento não possui uma propriedade que definimos, por exemplo na div número 2 abaixo vamos colocar um id chamada nao

1
2
3
            
<div class="container-not">
    <div class="box-not"1</div>
    <div class="box-not" id="nao">2</div>
    <div class="box-not">3</div>
</div>

.box-not:not(#nao){
    background-color: red;
}

has()

O has é o oposto, é quando um filtro aplicado quando o elemento possui uma propriedade, por exemplo vamos verificar se o container-has possui hover, se tiver coloque uma borda

1
2
3
            
<div class="container-has">
    <div class="box-has">1</div>
    <div class="box-has">2</div>
    <div class="box-has">3</div>
</div>

.box-has{
    width: 40px;
    height: 40px;
    border: solid;
    text-align: center;
    line-height: 250%;
    margin-top: 4px;
}

.box-has:hover{
    background-color: red;
}

.container-has:has(div:hover){
    border: solid
}

structural

Possui dois elementos para estilizar

  • root
  • nth-child(n)

root

Possui um peso maior que a própria tag na estilização

Nesse exemplo tudo ficaria vermelho, pois estou falando para todos os elementos ficarem com fundo vermelho

            
:root{
    background-color: red;
}

nth-child(n)

Podemos estilizar um elemento espefícico de um container, um filho desse container

1
2
3
            
.box-nth:nth-child(2){
    background-color: lightblue;
} //Nesse exemplo peguei o segundo filho do pai container-nth

Isso se por exemplo tivermos outro elemento entre o 1 e o 2, e quisermos selecionar ainda o mesmo elemento nesse caso o 2, temos que mudar o numero para 3, ele não segue por ordem de elemento

1
1.5
2
3
            
.box-nth-span:nth-child(3){
    background-color: lightblue;
} // aqui selecionamos o box numero 2, se quisessemos selecionar a span 1.5 mudariamos de .box-ntn-span:nth-chield(3) para span:nth-chield(2)
